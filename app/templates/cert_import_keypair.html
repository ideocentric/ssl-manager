Import Private Key + Certificate
Import an existing private key and its matching signed certificate. Use this to migrate certificates from an existing system (e.g. a directory-based workflow) into SSL Manager. The certificate file can be a single PEM, a PEM bundle (cert + intermediates), or a PKCS#7 (.p7b) file — intermediates are extracted automatically.

The private key and certificate must match. Import is aborted if the public keys do not correspond.
